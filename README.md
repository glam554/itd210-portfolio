# itd210-portfolio Portfolio Website- Grace Lam

## About This Project 
The Gentle Giants Project is a non-profit dedicated to helping the endangered whale shark species. It focuses on research, conservation, and education. The website provides information on the organization's efforts as well as information about whale sharks and ways visiters can support the group.

## Technologies Used
- HTML5
- CSS (Flexbox, Grid, Responsive Design)
- Javascript
